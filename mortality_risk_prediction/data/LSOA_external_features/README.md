# LSOA-level deprivation scores

Place these public Index of Multiple Deprivation score files here for the
Phase 0 harmonisation (`UKB_prepare.R`, `PMR_prepare.R`):

- `EIMD2010.xls` — English Index of Multiple Deprivation 2010 (LSOA 2001 scores).
- `WIMD2011.xls` — Welsh Index of Multiple Deprivation 2011 (LSOA 2001 scores).

Both are read by `UKB/UKB_harmonise.R` and `PMR/PMR_harmonise.R` to derive
`imd_decile`.

Sources: English Indices of Deprivation 2010 (MHCLG); Welsh Index of Multiple
Deprivation 2011 (Welsh Government).
